High-School Projects

Curriculum Vitae



Prototype of a Missile for Low-Intensity Bombing and Smoke Generation

Designed and fabricated a short-range, solid-propellant, air-to-surface missile controlled using wireless command guidance system for disintegration of riots and remote execution using unmanned combat aerial vehicle.

Paper-based Biosensing System for detection of Mastitis in Dairy Cattle

Devised a novel low-cost paper-based biosensor to detect increase in electrical conductance and count of somatic cells in milk using bibulous paper and current sensor for early prevention of mastitis in dairy cattle

Eyelink: A Low-cost Electrooculography based Human-Computer Interface

Developed an analog-to-digital circuit to acquire electrooculography data for controlling graphical user interface integrated with text-to-speech engine running on system on chip computer for argumentative and alternative communication using novel ‘Blink-to-Speak’ language.

iThink: An Electroencephalography based Wearable Brain-Computer Interface

Proposed a novel algorithm and trained a neural network to process EEG raw data to devise a wearable brain-computer interface for real-time brain-to-text, brain-to speech, and brain-to-command for miscellaneous applications.

P300 Speller for Brain-Computer Interfacing

Devised a brain-computer interface to interpret visually evoked p300 signal using support vector machine for spelling words in a custom word processor.

AudioOptic: A Computer Vision-based Wearable Visual Aid for Visually Challenged

Embedded array of optical sensors and processors in a headgear to implement a wearable visual aid working on parallel computing to detect, recognize and track objects, faces and text to translate them into speech for navigation and applications in real-time using concurrent visual processing algorithms.

MedAssist: Internet of Things based Automated Medicine Dispenser

Implemented a network of IoT-based automated medicine dispensers with real-time monitoring and configuration system for personal use and hospitals to assist patients with their medication schedule and to prevent medication non-adherence.

Unmanned Ground Vehicle with Vision for Remote Security and Surveillance

Implemented a continuous tracked, semi-autonomous unmanned ground vehicle with onboard six degrees of freedom robotics arm and depth-sensing for remote security and Surveillance.
